He tocado comandos basicos de linux como pueden ser:

pwd: que sirve para ver exactamente la ruta de directorios en la que estas.
ls: que sirve para listar los archivos que se encuentran en el directorio
actual.
whoami: que sirve para saber con que usuario te encuentras.

touch [nombre del archivo.extension] que sirve para crear un archivo.

nano [nombre del archivo + extension ] que sirve para abrir el archivo con 
el editor de texto nano

git init que sirve para inicializar un repositorio en github conectado 
al repositorio en el que te encuentras.

git config --global user.name // configura globalmente el nombre del usuario.
git config --global user.email // configura globalmente el email del usuario.
git config ls // muestra el user.name y el user.email
